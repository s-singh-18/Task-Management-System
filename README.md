# Task Management Application

A web-based application to efficiently manage your tasks.

## Features
* User authentication
* Task creation, editing, and deletion
* Responsive design
* Dark mode
* Stylish task cards

## Technologies Used
* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
